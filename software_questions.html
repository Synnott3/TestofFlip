Software Questions Page

FlipTheSwitch Common Software Solutions
Why is my operating system not functioning properly?

Solution

If the operating system or other software is either unresponsive or is acting up, then try restarting your computer and run a virus scan. To avoid having this happen, install reliable anti-virus software.
Why is my internet slow?

Solution

To improve your Internet browser performance, you need to clear cookies and Internet temporary files frequently. In the Windows search bar, type ‘%temp%’ and hit enter to open the temporary files folder.
Why won’t Windows boot up?

Solution

If you are having troubles booting Windows, then you may have to reinstall it with the Windows recovery disk.
My internet connection keeps dropping.

Solution

Dropped Internet connections can be very frustrating. Often the problem is simple and may be caused by a bad cable or phone line, which is easy to fix. More serious problems include viruses, a bad network card or modem, or a problem with the driver. Obviously if you are unsure, don’t try and fix it without any previous experience or know-how. With computers you could potentially make the problem ten times worse and even may not be able to recover the computer at all.
My Windows won’t update.

Solution

Windows update errors can occur due to a bunch of reasons. Causes include Windows piracy, core files missing, license issues, etc. The problem is, however, not that serious. But if you still want to have the latest software and security installed, you have to fix it. With an in-built error detection and troubleshooting, Windows has its own mechanism to fix this.
Why is my computer so slow?

Solution

If your computer hasn’t been turned off in a while, rebooting it should make it perform faster again. There might not be enough memory on the machine, try deleting unneeded files and applications to free up space. If too many programmes are running in the background, this can take up a lot of the computers resources so when you are done using an application or program, simply close it and the computer should speed up.
Why is my computer making strange noises?

Solution

A lot of strange noises is normally a sign of hardware failure or a noisy fan. Hardware normally makes strange noises right before they fail, so you should back-up any information before it does fail.
Why won’t my computer start?

Solution

A computer that suddenly turns off or has trouble turning on properly, the power supply might be failing. Check if the computer is plugged into the power supply properly and if that doesn’t work, try using other working devices to see if the power supply is adequate or not.
Why is my screen blank when my computer is on?

Solution

If the screen is blank, there might be an issue with the connection from the computer to the monitor. Check if the monitor is plugged into a power supply and the connection between the computer and the monitor.
My screen is frozen, what do I do?

Solution

If your computer is frozen, you will have to restart your computer and risk losing any unsaved documents and data. Freezing can be a sign of insufficient memory on the computer or corrupted or missing files or even malware or viruses. Press and hold the power button to turn your computer off and reboot it and get to work cleaning up the system to stop the computer freezing again in the future.
